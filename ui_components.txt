## Common UI Component Definitions

This document outlines common UI components for the translation website, including their descriptions, Tailwind CSS styling considerations, and relevant Lucide-react icon suggestions.

---

### 1. Button

*   **Component Name:** Button
*   **Description:** Used for triggering actions (e.g., submit form, open modal) or navigating to different parts of the application.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Base: `py-2 px-4 rounded-md font-semibold shadow-sm focus:outline-none focus:ring-2 focus:ring-offset-2 transition-colors duration-150 ease-in-out`
    *   Primary: `bg-indigo-600 text-white hover:bg-indigo-700 focus:ring-indigo-500`
    *   Secondary: `bg-gray-200 text-gray-700 hover:bg-gray-300 focus:ring-gray-400`
    *   Outline: `bg-transparent text-indigo-600 border border-indigo-600 hover:bg-indigo-50 focus:ring-indigo-500`
    *   Danger: `bg-red-600 text-white hover:bg-red-700 focus:ring-red-500`
    *   Disabled: `opacity-50 cursor-not-allowed`
    *   With Icon: `flex items-center space-x-2` (icon on left/right)
*   **Lucide-react Icons:** `LogIn` (Login), `LogOut` (Logout), `PlusCircle` (Add New), `Save` (Save), `Trash2` (Delete), `Edit3` (Edit), `Send` (Submit), `ArrowRight` (Next), `ArrowLeft` (Previous)

---

### 2. Card

*   **Component Name:** Card
*   **Description:** A container for grouping related information, often used to display summaries, previews, or distinct sections of content.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Base: `bg-white shadow-lg rounded-lg overflow-hidden`
    *   Padding: `p-4` or `p-6`
    *   Header/Footer: `px-4 py-3 bg-gray-50 border-b border-gray-200` (for header), `px-4 py-3 bg-gray-50 border-t border-gray-200` (for footer)
    *   Title: `text-lg font-semibold text-gray-800`
    *   Body: `text-gray-600`
    *   Interactive: `hover:shadow-xl transition-shadow duration-200`
*   **Lucide-react Icons:** `FileText` (Document Preview), `Users` (User Profile Card), `Briefcase` (Project Card)

---

### 3. Modal

*   **Component Name:** Modal (Dialog)
*   **Description:** A dialog box/popup window that is displayed on top of the current page, used for important messages, forms, or tasks that require user focus.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Overlay: `fixed inset-0 bg-black bg-opacity-50 flex items-center justify-center z-50 transition-opacity duration-300`
    *   Container: `bg-white rounded-lg shadow-xl p-6 m-4 max-w-lg w-full transform transition-all duration-300 ease-out` (add `scale-95 opacity-0` for entry/exit transitions)
    *   Title: `text-xl font-semibold text-gray-900 mb-4`
    *   Body: `text-gray-700 mb-6`
    *   Footer/Actions: `flex justify-end space-x-3`
    *   Close Button: `absolute top-3 right-3 text-gray-400 hover:text-gray-600`
*   **Lucide-react Icons:** `X` (Close), `AlertTriangle` (Warning Modal), `CheckCircle` (Success Modal), `Info` (Info Modal)

---

### 4. TextInput (Input Field)

*   **Component Name:** TextInput
*   **Description:** Allows users to enter and edit text or numerical data.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Base: `block w-full px-3 py-2 border border-gray-300 rounded-md shadow-sm focus:outline-none focus:ring-indigo-500 focus:border-indigo-500 sm:text-sm`
    *   Disabled: `bg-gray-100 cursor-not-allowed text-gray-500`
    *   Error State: `border-red-500 text-red-600 focus:ring-red-500 focus:border-red-500`
    *   With Icon (Left/Right): `relative`, icon positioned absolutely within.
    *   Label: `block text-sm font-medium text-gray-700 mb-1`
    *   Helper Text/Error Message: `mt-1 text-xs text-gray-500` or `mt-1 text-xs text-red-600`
*   **Lucide-react Icons:** `User` (Username), `Mail` (Email), `Lock` (Password), `Search` (Search Input), `CalendarDays` (Date Input)

---

### 5. Dropdown (Select)

*   **Component Name:** Dropdown (Select)
*   **Description:** Allows users to choose one option from a predefined list.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Base: `block w-full pl-3 pr-10 py-2 border border-gray-300 rounded-md shadow-sm focus:outline-none focus:ring-indigo-500 focus:border-indigo-500 sm:text-sm bg-white text-gray-700`
    *   Appearance: `appearance-none` (to hide default system arrow, often paired with a custom arrow icon)
    *   Custom Arrow Wrapper: `pointer-events-none absolute inset-y-0 right-0 flex items-center px-2 text-gray-700`
    *   Disabled: `bg-gray-100 cursor-not-allowed text-gray-500 opacity-70`
    *   Options: Typically styled by the browser or a custom dropdown library.
    *   Label: `block text-sm font-medium text-gray-700 mb-1`
*   **Lucide-react Icons:** `ChevronDown` (Dropdown Arrow), `Languages` (Language Selector), `Filter` (Filter Dropdown)

---

### 6. Tabs

*   **Component Name:** Tabs
*   **Description:** Allows users to switch between different views or sections of content within the same context.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Container: `border-b border-gray-200`
    *   Tab List: `flex -mb-px space-x-4` (or `space-x-8`)
    *   Tab Item (Base): `py-2 px-1 whitespace-nowrap cursor-pointer text-gray-500 hover:text-gray-700 hover:border-gray-300`
    *   Tab Item (Active): `border-b-2 border-indigo-500 text-indigo-600 font-medium`
    *   Tab Item (Inactive): `border-b-2 border-transparent`
    *   Tab Panel: `mt-4` (or `py-4`)
*   **Lucide-react Icons:** `List` (List View Tab), `Settings` (Settings Tab), `Info` (Details Tab)

---

### 7. Alert

*   **Component Name:** Alert
*   **Description:** Displays important messages to the user, such as success, warning, error, or informational notifications.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Base: `p-4 rounded-md flex items-start space-x-3`
    *   Success: `bg-green-50 text-green-700` (Icon: `text-green-400`)
    *   Error: `bg-red-50 text-red-700` (Icon: `text-red-400`)
    *   Warning: `bg-yellow-50 text-yellow-700` (Icon: `text-yellow-400`)
    *   Info: `bg-blue-50 text-blue-700` (Icon: `text-blue-400`)
    *   Icon Container: `flex-shrink-0`
    *   Text Container: `flex-1`
    *   Title: `font-medium`
    *   Description: `text-sm`
    *   Dismiss Button (Optional): `ml-auto pl-3 -mx-1.5 -my-1.5`
*   **Lucide-react Icons:** `CheckCircle2` (Success), `XOctagon` (Error), `AlertTriangle` (Warning), `Info` (Info), `X` (Dismiss)

---

### 8. Tag (or Badge)

*   **Component Name:** Tag (or Badge)
*   **Description:** Used for labeling, categorizing, or highlighting items with keywords or status indicators.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Base: `inline-flex items-center px-2.5 py-0.5 rounded-full text-xs font-medium`
    *   Primary: `bg-indigo-100 text-indigo-800`
    *   Secondary: `bg-gray-100 text-gray-800`
    *   Success: `bg-green-100 text-green-800`
    *   Warning: `bg-yellow-100 text-yellow-800`
    *   Error: `bg-red-100 text-red-800`
    *   Subtle: `bg-gray-50 text-gray-600 ring-1 ring-inset ring-gray-500/10` (like GitHub labels)
    *   With Dot: `flex items-center space-x-1.5` (dot before text)
*   **Lucide-react Icons:** `Tag` (General Tag), `Circle` (Status Dot for Tag), `Archive` (Archived Tag)

---

### 9. FileUploader

*   **Component Name:** FileUploader
*   **Description:** Allows users to select and upload one or more files from their local system.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Container (Dropzone): `flex justify-center items-center w-full px-6 py-10 border-2 border-gray-300 border-dashed rounded-md cursor-pointer hover:border-indigo-500 bg-gray-50 hover:bg-indigo-50 transition-colors`
    *   Text/Icon Area: `text-center`
    *   Upload Icon: `mx-auto h-12 w-12 text-gray-400`
    *   Instruction Text: `mt-2 block text-sm text-gray-600`
    *   File Input (Hidden): `hidden` (triggered by JavaScript)
    *   File List (Preview): `mt-4 space-y-2`
        *   File Item: `flex justify-between items-center p-2 border rounded-md bg-white`
        *   File Name: `text-sm text-gray-700`
        *   Progress Bar (Optional): `w-full bg-gray-200 rounded-full h-1.5 dark:bg-gray-700` with an inner `bg-indigo-600` bar.
        *   Remove Button: `text-red-500 hover:text-red-700`
*   **Lucide-react Icons:** `UploadCloud` (Upload Icon), `File` (File Icon), `Trash2` (Remove File)

---

### 10. Table

*   **Component Name:** Table
*   **Description:** Displays structured data in rows and columns, suitable for presenting lists of items with multiple attributes.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Wrapper: `overflow-x-auto shadow-md rounded-lg`
    *   Table: `min-w-full divide-y divide-gray-200 bg-white`
    *   Thead: `bg-gray-50`
    *   Th: `px-6 py-3 text-left text-xs font-medium text-gray-500 uppercase tracking-wider`
    *   Tbody: `bg-white divide-y divide-gray-200`
    *   Tr: `hover:bg-gray-50 transition-colors`
    *   Td: `px-6 py-4 whitespace-nowrap text-sm text-gray-700`
    *   Action Cell (e.g., for Edit/Delete buttons): `text-right space-x-2`
*   **Lucide-react Icons:** `ArrowUpDown` (Sortable Column Header), `Edit` (Edit Row Action), `Trash` (Delete Row Action)

---

### 11. Pagination

*   **Component Name:** Pagination
*   **Description:** Allows users to navigate through a large set of items موزعة across multiple pages.
*   **Key Visual Elements/Styling (Tailwind CSS):**
    *   Container: `flex items-center justify-between border-t border-gray-200 bg-white px-4 py-3 sm:px-6 mt-4` (or simple `flex justify-center space-x-1 mt-4`)
    *   Button (Base): `relative inline-flex items-center px-4 py-2 text-sm font-medium rounded-md`
    *   Button (Number): `text-gray-700 bg-white hover:bg-gray-50`
    *   Button (Active): `z-10 bg-indigo-50 border-indigo-500 text-indigo-600`
    *   Button (Prev/Next): `text-gray-500 hover:bg-gray-50`
    *   Button (Disabled): `text-gray-300 cursor-not-allowed`
    *   Ellipsis: `px-4 py-2 text-sm font-medium text-gray-700`
    *   Info Text (Optional): `text-sm text-gray-700` (e.g., "Showing 1 to 10 of 57 results")
*   **Lucide-react Icons:** `ChevronLeft` (Previous), `ChevronRight` (Next), `ChevronsLeft` (First), `ChevronsRight` (Last)
